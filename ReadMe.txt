"ConferenceTrackManagement" Project
===================================

/* How to run and compile this Project? */
Add this project to your Visual Studio Code.
And finally, compile (Build Project).
Please don't forget to put the unscheduled talk list as a text file (Input.txt) in the project bin path: /ConferenceTrackManagement/bin/Debug/netcoreapp1.1/
The output, which is the scheduled Conference Program as a text file (Output.txt) will be avaiable in the same bin directory path, after the program runs properly.
